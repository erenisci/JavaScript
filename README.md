# JavaScript Learning Repository

This repository contains my exercises, projects, and capstone projects completed while studying JavaScript through Jonas Schmedtmann's course. The course covers fundamental JavaScript concepts, DOM manipulation, functions, operators, arrays, OOP (Object-Oriented Programming), asynchronous programming etc.

## Course Information

I completed this course to build a strong foundation in JavaScript and learn to apply it effectively in real-world projects:

- **Course Name**: The Complete JavaScript Course 2024: From Zero to Expert!
- **Instructor**: Jonas Schmedtmann
- **Course Link**: [Udemy Course Link](https://www.udemy.com/course/the-complete-javascript-course/) (This course is continuously updated.)
